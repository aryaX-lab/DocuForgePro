## ✨ Features

### 🛠️ Core Tools
- 🔗 **Merge** - ترکیب چندین PDF
- ✂️ **Split** - جداسازی با حالت‌های مختلف
- 🔄 **Rotate** - چرخش 90°, 180°, 270°
- 📑 **Page Manager** - مدیریت کامل صفحات
- ✍️ **Annotate** - علامت‌گذاری و امضا
- 🔒 **Security** - رمزگذاری و حذف اطلاعات
- 📉 **Compress** - فشرده‌سازی هوشمند
- 🎯 **Optimize** - بهینه‌سازی برای وب یا چاپ

### 🎨 UI Features
- 🌙 Dark mode و 🖥️ Presentation mode
- 📊 Analytics dashboard
- 💾 Auto backup system
- 🌐 Bilingual (FA/EN)
- 🎨 8 color themes
- ⚡ Drag & Drop

### 🏗️ Technical
- 💚 Health Check System
- ⚡ Performance Monitor
- 🔄 Auto-Recovery
- 📝 Advanced logging

## 🚀 Installation

### Installer
1. `DocuForge-Pro-Setup-v2.0.0.exe` را دانلود کن
2. اجرا کن و مراحل نصب را دنبال کن
3. از Start Menu اجرا کن

### Portable
1. `DocuForge-Pro-Portable-v2.0.0.zip` را دانلود کن
2. در پوشه دلخواه Extract کن
3. `DocuForge Pro.exe` را اجرا کن
4. نیازی به نصب نیست!
